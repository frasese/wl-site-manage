Importar material ou participantes de outro site

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Para qualquer curso ou site de projeto que você cria, você pode importar um instrutor de conteúdo gerado (por exemplo, avisos, as atividades, as configurações da ferramenta) de um ou mais sites que você possui. Material submetido, como atividades cumpridas, mensagens do fórum, ou notas, não podem ser importadas usando os seguintes métodos, você deve baixar e salvar esse material manualmente.

Nota: Você não pode importar os testes do Teste Original e uma ferramenta de pesquisa usando este método. Em vez disso, ver [documento disponível: akti].

Importar material de outro site

Para importar material de outro site:

  1. No menubar do site de destino, clique em Site Info.

  2. No topo da tela do Site Info, clique em Importar do Site.

  3. Na tela de Importação de Dados, escolha uma das seguintes opções:

    • Eu gostaria de substituir os meus dados: . Utilize esta opção se você deseja que os dados importados substituem os dados existentes. Este método só permite que você selecione um site para importar os dados.
    • Eu gostaria de mesclar os meus dados: Utilize esta opção se você deseja que os dados importados mesclem com os dados existentes. Este método permite que você selecione vários sites para importar os dados.
    • Eu gostaria de mesclar o meu(s) usuário (s): Use esta opção se você deseja importar a lista dos participantes de outro site.

  4. Selecione a caixa próxima ao site que contém o material de origem. Em seguida, clique em Continuar.

  5. Se for o caso, sob o nome do site (s) selecionado, marque a caixa ao lado de cada ferramenta que contem o material de origem que você deseja importar. Se uma caixa estiver cinza, significa que a ferramenta correspondente não está habilitada nesse site. Em seguida, clique em Finalizar.

Notas

  • Itens salvos como rascunhos no site de origem serão importados como rascunhos, o que significa que eles não são imediatamente visíveis pelos participantes. Para torná-los visíveis, você deve publicá-los. Para obter instruções, consulte Postar uma atividade previamente salva como um rascunho.

  • Quando as atividades são importadas de outro site, a opção Adicionar para Quadro de Notas é selecionada por padrão. Se tal atividade é atualizada e salva, uma entrada no Quadro de Notas correspondente será criada, que pode ou não ser o que foi planejada. Para associar o item a uma entrada existente no Quadro de Notas, ao invés de criar um item novo no Quadro de Notas a partir da ferramenta Atividades, selecione Associar com entrada existente no Quadro de Notas e escolha a entrada do Quadro de Notas desejada antes de postar. Para mais informações sobre as diferenças entre as duas opções, consulte Em Atividades, qual método devo usar para vincular minha atividade ao Quadro de Notas?

  • Quando você importa as configurações do Quadro de Notas, pode ser possível importar um item com um do valor de ponto do item do Quadro de Notas inferior ou igual a 0 (zero), no entanto, se você tentar editar um item deste tipo, você receberá uma mensagem de erro.
