Duplicar um site

Quando você duplica um site de curso, você cria uma cópia do site que não é exatamente como o original. O site duplicado não irá incluir qualquer aluno ou material; apenas o conteúdo postado por instrutores será incluído.

Quando você duplica um site de projeto, você cria uma cópia do site que é exatamente como o original. Todos os materiais dos participantes inscritos serão incluídos.

Para duplicar seu site:

  1. Quando você tiver o site aberto, a partir do menubar, clique em Site Info.

  2. Clique em Duplicar Site.

  3. No campo "Título do Site:", digite um nome para o site duplicado, selecione o Termo Academico, e clique em Duplicar.

    Nota: Se você não tem direitos suficientes para criar um site, você receberá a mensagem, "Alerta: Você não tem permissão para criar um novo site."

  4. Clique em Finializar.
